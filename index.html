Uncorked

와인 경험을 디지털 여정으로 번역한 서비스

문제
오프라인 와인 경험은 온라인에서 맥락이 사라진다.
해결
취향 기반 탐색 흐름과 스토리형 카드 UI를 설계했다.
결과
탐색 시간이 줄고 저장/재방문 흐름이 자연스러워졌다.

가장 어려웠던 결정: 감성적 브랜딩과 검색 효율 사이의 균형.

EZAir

복잡한 항공 탐색을 단순화한 예약 경험

문제
항공권 필터가 복잡해 사용자가 초기에 이탈했다.
해결
핵심 필터 우선 노출과 단계형 검색 플로우를 도입했다.
결과
검색 완료율이 올라가고 비교 시간이 단축됐다.

가장 어려웠던 결정: 옵션 다양성과 초기 화면 단순성의 트레이드오프.

PortfoliOh

결과

가장 어려웠던 결정: 기능 실험성과 정보 전달 밀도의 균형.
